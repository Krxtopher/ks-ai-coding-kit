# System Prompts & Custom Instructions

Tool-specific system prompts and custom instruction files for various AI coding tools.

## What Are These?

Different AI coding tools use different files for system-level instructions:

| Tool | File | Location |
|------|------|----------|
| Claude Code | `CLAUDE.md` | Project root |
| Codex | `AGENTS.md` or `codex.md` | Project root |
| GitHub Copilot | `.github/copilot-instructions.md` | Project root |
| Cursor | `.cursorrules` | Project root |

This directory organizes reusable prompts by target tool.

## Installation

Copy the prompt file(s) for your tool into your project root:

```bash
cp prompts/claude-code/CLAUDE.md /path/to/your/project/
```

Some tools may require the file in a specific subdirectory — check the tool-specific README for details.
